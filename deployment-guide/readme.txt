A brief information on how the code is deployed from a lower environment to higher environment at org level.

For Azure Data Factory Deployment:
1. A workflow is created under 'Actions' tab above with relevant pipelines, global parameters, linked services and other resources.
2. When deploying from dev to QA, a developer/engineer will navigate to the workflow and run it manucally from GitHub UI.

For Power BI deployment:
1. BI engineer/analyst to update the reports on PBI desktop and publish to relevant workspaces.

For SharePoint updates:
1. BI engineer created a step-by-step guide document for the business team to utilize to make updates in SharePooint site UI/portal.
